Installation Guide - Manual

Installation guide for Live Chat 3.

Please follow all the steps carefully.

Step by Step
  • Upload the content of this folder into your domain or sub domain. !!! Important !!! Do not use a sub folder!
  • Make sure all permissions are correct. Folders (0755), Files (0644)
  • Create two databases and database users, make sure to remeber the informations
  • Go to http://www.yourdomain.com/install/install.php and follow the installation wizard.
  • After successful installation delete the install directory!
